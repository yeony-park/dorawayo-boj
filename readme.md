# 돌아와요 BOJ

**🔗 https://yondori.com/dorawayo-boj**

---

## 배경

2010년 3월 개설된 백준 온라인 저지(BOJ)가 **2026년 4월 28일을 끝으로 서비스를 종료**합니다.
운영자 최백준 님이 16년간 1인 운영해온 서비스로, 종료 공지([원문](https://www.acmicpc.net/board/view/165799))에 따르면 서비스 종료 시점에 문제·제출 기록·대회 정보 등을 제외한 모든 데이터가 삭제될 예정입니다.

이에 앞서 공개된 문제들을 아카이빙하여 계속 열람할 수 있도록 본 서비스를 구현하였습니다.

> **유의사항**
> - 테스트 케이스는 공개된 바 없어 문제 본문만 수록되어 있습니다.
> - 저작권 또는 기타 문제 발생 시 해당 콘텐츠는 즉시 삭제합니다.

---

## 기능

- **문제 목록** : 수록된 전체 문제를 번호 순으로 표시
- **티어 필터** : Bronze / Silver / Gold / Platinum / Diamond / Ruby / Unrated 별 필터링
- **검색** : 문제 번호 또는 제목으로 검색
- **문제 상세** : 문제 설명, 입출력 형식, 예제 입출력(복사 버튼 포함), 태그, 출처 표시
- **페이지네이션** : 페이지당 50문제

---

## 사용 방법

1. https://yondori.com/dorawayo-boj 접속
2. 목록에서 문제를 클릭하면 상세 페이지로 이동
3. 티어 드롭다운 또는 검색창으로 원하는 문제 필터링
4. 예제 입출력은 복사 버튼으로 바로 복사 가능
